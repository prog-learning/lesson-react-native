# Lesson React Native

React と TypeScript の知見を活かして Native アプリ開発に入門する

## Expo に deploy して Expo Go で動かせるようにする

Expo のアカウントを作成しておく

```sh
expo build:ios
```

最初に名前を入力

```sh
✔ Choose the build type you would like: › simulator
```

と選択すること

Expo のアカウントでログイン
